# RBP4 - Three Simple Commands ⚡

Everything is ready! Just use one of these three commands based on your setup:

---

## 🚀 QUICK REFERENCE

### Build (one-time)
```bash
./RBP4/build_rbp4.sh
```

### Then Pick One:

**1️⃣  WITH WINDOW** (see live detection)
```bash
./RBP4/run_rbp4_headless.sh
```
Use when: You have a monitor attached to RPi

**2️⃣  WITHOUT WINDOW** (headless/SSH)
```bash
./RBP4/run_rbp4.sh
```
Use when: Connecting via SSH, no monitor

**3️⃣  SIMULATION** (no hardware)
```bash
./RBP4/run_rbp4_sim.sh
```
Use when: Testing at home without camera/BLE

---

## ✅ That's It!

- Car connects automatically
- Detects red car automatically
- Follows autonomously
- Press **Ctrl+C** to stop

**Everything works exactly like the main project!** 🎉
